\doxysubsection{Detailed Description}
A CCS tree tuner defines an ask and tell interface to optimize an objective space (see \mbox{\hyperlink{objective__space_8h}{objective\+\_\+space.\+h}}) given a tree space (see \mbox{\hyperlink{tree__space_8h}{tree\+\_\+space.\+h}}). 

The tuner will propose tree configurations (see \mbox{\hyperlink{tree__configuration_8h}{tree\+\_\+configuration.\+h}}) and the user will return tree evaluations (see \mbox{\hyperlink{tree__evaluation_8h}{tree\+\_\+evaluation.\+h}}).
