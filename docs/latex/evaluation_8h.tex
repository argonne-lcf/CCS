\doxysubsection{Detailed Description}
An evaluation is a binding (see \mbox{\hyperlink{binding_8h}{binding.\+h}}) over an objective space (see \mbox{\hyperlink{objective__space_8h}{objective\+\_\+space.\+h}}) given a specific configuration (see \mbox{\hyperlink{configuration_8h}{configuration.\+h}}). 

Successful evaluations over the same objective space are weakly ordered by their objective values. Evaluation that have failed must report a result code different than CCS\+\_\+\+RESULT\+\_\+\+SUCCESS.
