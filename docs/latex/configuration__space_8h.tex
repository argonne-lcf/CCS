\doxysubsection{Detailed Description}
A configuration space is a context (see \mbox{\hyperlink{context_8h}{context.\+h}}) defining a set of parameters. 

Configuration space also offer as constraints system to describe conditional parameter activation as well as forbidden parameter configurations. Sampling distributions of parameters can be specified.
