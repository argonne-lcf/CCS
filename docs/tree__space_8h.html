Detailed Description

A CCS tree space defines an search space over a tree.

Tree spaces can be sampled to obtain tree configurations. Configurations con point to unknown children (if they exist). This is where static tree spaces and dynamic tree spaces differ: in a static tree space, the user is responsible for modifying the tree before evaluating these configuration values or node; in a dynamic tree space, when querying a configuration validity, node or values, a callback will be invoked to define the missing children on the configuration path. Sampling a dynamic tree space, by itself, does not modify the tree.
