Detailed Description

Hyperparameters are parameters the when grouped together define a tuning context (see context.h).

Most hyperparameters can be sampled according to a distribution's dimension (see distribution.h).

Enumeration Type Documentation

◆ ccs_hyperparameter_type_e

enum ccs_hyperparameter_type_e

CCS hyperparameter types.

Enumerator
CCS_HYPERPARAMETER_TYPE_NUMERICAL 

A numerical hyperparameter, over intergers or floating point values.

CCS_HYPERPARAMETER_TYPE_CATEGORICAL 

A categorical hyperparameter.

A set of values with no intrinsic ordering between its elements.

CCS_HYPERPARAMETER_TYPE_ORDINAL 

An ordinal hyperparameter.

A set of values with an intrinsic ordering between its elements.

CCS_HYPERPARAMETER_TYPE_DISCRETE 

A discrete hyperparameter.

A set of numerical values.

CCS_HYPERPARAMETER_TYPE_STRING 

A string hyperparameter.

Cannot be sampled.

CCS_HYPERPARAMETER_TYPE_MAX 

Guard.

CCS_HYPERPARAMETER_TYPE_FORCE_32BIT 

Try forcing 32 bits value for bindings.
