Detailed Description

A features is a binding (see binding.h) over a features space (see features_space.h).

ccs_error_t ccs_create_features ( ccs_features_space_t  features_space,
size_t  num_values,
ccs_datum_t *  values,
ccs_features_t *  features_ret 
)

Create a new instance of a features on a given features space.

If no values are provided the values will be initialized to CCS_NONE.

Parameters
[in]features_space
[in]num_valuesthe number of provided values to initialize the features instance
[in]valuesan optional array of values to initialize the features
[out]features_reta pointer to the variable that will hold the newly created features
Returns
CCS_SUCCESS on success
CCS_INVALID_OBJECT if features is not a valid CCS features space
CCS_INVALID_VALUE if features_ret is NULL; or if values is NULL and num_values is greater than 0; or if the number of values provided is not to the number of hyperparameters in the features space
CCS_OUT_OF_MEMORY if there was a lack of memory to allocate the new features

Define a strict ordering of features instances.

Parameters

[in]other_featuresthe second features
[out]cmp_retthe pointer to the variable that will contain the result of the comparison. Will contain -1, 0, or 1 depending on if the first features is found to be respectively lesser than, equal, or greater then the second features
Returns
CCS_SUCCESS on success
