Detailed Description

A features space is a context (see context.h) defining a set of parameters.

Check that a features is a valid in a features space.

Parameters
[in]features_space
[in]features
[out]is_valid_reta pointer to a variable that will hold the result of the check. Result will be CCS_TRUE if the features is valid. Result will be CCS_FALSE if an parameter value is not a valid value for this parameter;
Returns
CCS_RESULT_SUCCESS on success
CCS_RESULT_ERROR_INVALID_OBJECT if features_space is not a valid CCS features space; or if features is not a valid CCS features
CCS_RESULT_ERROR_INVALID_FEATURES if features is not associated to the features space; or if the number of values contained in features is not equal to the number of parameters in the features space

)

Validate that a given value at the given index is valid in a features space, and return a sanitized value.

Parameters
[in]features_space
[in]indexthe index of the value in the features_space
[in]valuethe datum to validate
[out]value_reta pointer that will contain the validated value. If value is a string value_ret will contain a non transient string.
Returns
CCS_RESULT_SUCCESS on success
CCS_RESULT_ERROR_INVALID_OBJECT if features_space is not a valid CCS features space
CCS_RESULT_ERROR_OUT_OF_BOUNDS if index is greater than the number of parameters in features_space
CCS_RESULT_ERROR_OUT_OF_MEMORY if there was a lack of memory while memoizing a string
CCS_RESULT_ERROR_INVALID_VALUE if the value did not validate or if value_ret is NULL
